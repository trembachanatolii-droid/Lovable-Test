# Free Professional Images Download Guide

This guide provides free, royalty-free images from Pexels and Pixabay to replace Unsplash images. All images are:
- **Free for commercial use**
- **No attribution required**
- **No people** (as requested)

## How to Download

1. Click the link to open the image page
2. Click "Free Download" button
3. Select "Large" or "Original" size (1920px width recommended)
4. Save to `/public/images/` folder with the suggested filename

## After Downloading Images

Once you download images to `/public/images/`, I can help update the code references. The update pattern will be:

**Before:**
```tsx
src="https://images.unsplash.com/photo-XXXXXXXXX?q=80&w=1920&auto=format&fit=crop"
```

**After:**
```tsx
src="/images/port-aerial-cargo-ship.jpg"
```

This will improve:
- **LCP (Largest Contentful Paint)**: 1-2 seconds faster
- **Reliability**: No external dependencies
- **Privacy**: No third-party tracking
