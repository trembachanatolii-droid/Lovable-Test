# Images Directory

## Tariff Classification Hero Background

Place your container port/harbor night image here with the filename:
**tariff-hero-port-harbor.jpg**

The image will be used as the background for the Tariff Classification hero section on the homepage carousel.

### Recommended Image Specifications:
- Format: JPG or PNG
- Minimum Resolution: 1920x1080
- Aspect Ratio: 16:9 or wider
- File Size: Optimized for web (< 500KB recommended)

### Current Usage:
- HomePage carousel - First slide (Tariff Classification)
